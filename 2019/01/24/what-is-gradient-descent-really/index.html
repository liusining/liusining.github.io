10分钟深入理解梯度下降，面试再也不怕了

Posted on 2019-01-24 | Edited on 2019-02-14 |
原来这就是你心心念的「梯度下降」

wikipedia 的解释是：

Gradient descent is a first-order iterative optimization algorithm

如果有同学不熟悉数学名词的英语表达（其实我也是刚查的），那么有这么几个看似常见，但是含义却很不一样的单词需要注意：

  1. order 是阶，first-order 就是一阶（不知道这有没有让你想起一阶导数）
  2. iterative 是迭代，或者叫递归，对于一个优化过程来说，可以特指先猜一个结果，然后逐次优化，直到到达某个结束条件

接下来就是 optimization，可能大家都知道这是（最）优化的意思，但更重要的问题是我们要优化什么？别着急，wikipedia 的解释中还有后半句:

Gradient descent is a first-order iterative optimization algorithm for finding the minimum of a function
